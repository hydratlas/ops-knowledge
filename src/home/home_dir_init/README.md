# home_directory ロール

ホームディレクトリの所有者・権限設定と、スケルトンファイルのコピーを行うロールである。

## 変数

| 変数名             | 必須 | デフォルト | 説明                           |
| ------------------ | ---- | ---------- | ------------------------------ |
| `home_directories` | Yes  | `[]`       | ホームディレクトリ設定のリスト |

### home_directories の構造

| キー    | 必須 | デフォルト | 説明                         |
| ------- | ---- | ---------- | ---------------------------- |
| `path`  | Yes  | -          | ホームディレクトリのパス     |
| `user`  | Yes  | -          | 所有者ユーザー名             |
| `group` | No   | `user`の値 | 所有グループ名               |
| `mode`  | No   | `0700`     | ディレクトリのパーミッション |

## 動作

1. 指定されたパスの所有者・グループ・パーミッションを設定する
2. `/etc/skel/`の内容をホームディレクトリにコピーする（既存ファイルは上書きしない）
